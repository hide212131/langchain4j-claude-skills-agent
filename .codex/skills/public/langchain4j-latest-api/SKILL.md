# LangChain4j Latest API

## 概要

LangChain4jの最新APIを前提に、LLMアクセスに関係するコードの生成・修正を行うための手順を示す。
必要に応じてローカルの参照リポジトリから最新仕様を確認する。

## 基本手順

1. 依頼内容がLLMアクセス関連か、LangChain4jのAPI参照を求める内容かを確認する。
2. 最新APIの仕様を確認するため、必要に応じて以下を参照する。
   - ソースコード: `../langchain4j` と `../langchain4j-examples/`
   - ドキュメント: `../langchain4j/docs`
3. 既存コードの修正時は、利用中のAPIが最新仕様に合致しているかを特定する。
4. 最新APIに合わせてコードを生成・修正し、必要に応じて差分の根拠を明記する。
5. API差分や前提が不明な場合は、確認事項を短く列挙する。

## 期待される出力の方針

- 生成したコードは最新APIに準拠させること。
- 参考にしたAPIの場所や理由が重要な場合は短く付記すること。
- 依頼が曖昧なときは、必要最小限の質問に留めること。
